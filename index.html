My Projects

Meet App

A React app, designed as a PWA to retrieve meetups. Limiting number of events and providing suggestions are also possible. The code was developed in the sense of a BDD and TDD approach (use of Jest and Cucumber). The web application is used to see the upcoming events which are set up in Google Calendar API.



Project on Github Hosted project

myFlix-Client (React)

An app hosted on Netlify, connected to the myFlix API I created, which retrieves film data from the MongoDB database. It was designed using React-Bootstrap. The website is designed to show the retrieved film data and enable a user profile section. Functions such as logging in, logging out, registering and de-registering users are also integrated.


Project on Github Hosted project

myFlix-Client (Angular)

A single-page, responsive movie app built with Angular, routing and several interface views. It was formed using Angular Material, and connected to the myFlix API I created, which retrieves film data from my MongoDB database. The website was designed to show the retrieved film data and enable a user profile section. Functions such as logging in, logging out, registering and de-registering users are also integrated.
